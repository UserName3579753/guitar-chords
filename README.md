# Guitar Chords App

A PWA (Progressive Web App) for managing guitar chord diagrams.

## Features
- Google login
- Add, view, edit and delete chord diagrams
- 6 strings, 5 frets with proper nut at top
- Drag finger dots onto the diagram
- Cloud storage via Firebase Firestore
- Works as a home screen app on iPhone

## Setup

1. Upload `icon.png` (the monkey image) to the repository root — rename the file to `icon.png`
2. Connect the repository to Netlify
3. Open the Netlify URL in Safari on iPhone
4. Tap Share → "Add to Home Screen"

## Tech
- Vanilla HTML/CSS/JS (no build step needed)
- Firebase Auth + Firestore
- PWA with manifest
